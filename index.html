Figpack URL References

This repository maintains a collection of references to Figpack URLs found in public GitHub repositories. The data is automatically updated daily through a GitHub Actions workflow.

The JSON file contains records with the following structure:

{
  "repo": "owner/name",
  "file": "relative/path/to/file.md",
  "url": "https://figures.figpack.org/..."
}
